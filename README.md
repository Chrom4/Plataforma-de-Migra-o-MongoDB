# 🔄 Plataforma de Migração MongoDB

Uma plataforma web focada em automação de infraestrutura, projetada para orquestrar e executar migrações de bancos de dados MongoDB entre diferentes servidores e ambientes de clientes de forma segura e visual.

## 🚀 Funcionalidades Principais
* **Migração Automatizada:** Transferência de coleções inteiras entre clusters e servidores distintos sem intervenção manual no terminal.
* **Transferência de Alto Volume:** Arquitetura otimizada para suportar a manipulação e transferência de grandes volumes de dados de forma estável.
* **Garantia de Integridade:** Validação de dados pós-migração para garantir que as coleções de destino sejam réplicas exatas da origem.
* **Interface de Controle:** Painel web para acompanhamento do status das migrações em tempo real.

## 💻 Tecnologias Utilizadas
**Frontend:**
* React
* Vite

**Backend & Infraestrutura:**
* Node.js
* MongoDB (Drivers nativos / Mongoose)
